# 万花阁-小程序

## 介绍
* 基于wepy开发的小程序
* 史上最简单、最纯粹的看美女微信小程序

## 已实现功能
* 随机从数据库获取10幅精选图集
* 根据评分排行榜显示图集
* 本地收藏图集

## 安装使用
* 在小程序后台配置：https://www.nichuiniu.cn为请求的域名地址
* 克隆到本地，通过微信小程序开发软件打开即可
